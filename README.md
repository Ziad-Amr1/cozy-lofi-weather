# Cozy Lofi Weather

A cozy anime-inspired weather web app built with Node.js, Express, and Handlebars.

The app allows users to search for weather data using either a city name or geographic coordinates, with a modern glassmorphism UI, animated transitions, and responsive design.

---

## Features

* Search weather by city name
* Search weather by latitude & longitude
* Real-time weather data
* Toast notifications system
* Animated glassmorphism UI
* Responsive mobile-friendly layout
* Dynamic weather rendering with JavaScript
* Lofi / anime inspired aesthetic

---

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/cozy-lofi-weather.git
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
WEATHER_API=YOUR_API_KEY
GEO_CODE_TOKEN=YOUR_MAPBOX_TOKEN
```

Run the project:

```bash
npm run dev
```
